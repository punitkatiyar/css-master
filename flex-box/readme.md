# flex Box 

flex is used for creating flexible layout structure with out using float and position

> display: flex;

## flex-direction

> row (default)

> row reverse

> column

> column reverse

## Flex Wrap

> nowrap (default)

> wrap

> wrap reverse

## justify-content (align)

> flex-start (default)

> flex-end

> start

> end

> left

> right

> center

> space-between

> space-around

> space-evenly

## align-items (v-align)

> stretch (default)

> start

> end
